### Where Clause Operators

| Operator      | Description |
| ----------- | ----------- |
| =      | Equal       |
| <>   | Not equal. Note: in some versions of SL this operator may be written as !+        |
| > | Greater than |
| < | Less than |
| >= | Greater than or equal |
| <= | Less than or equal |
| BETWEEN | Between an inclusive range |
| IS NULL | Is a null value |
